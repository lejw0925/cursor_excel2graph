# 数据可视化工具

一个纯前端的数据可视化工具，支持Excel/CSV文件上传、数据预览和多种交互式图表生成。

## 功能特点

### 📊 数据处理
- 支持上传 Excel (.xlsx, .xls) 和 CSV 文件
- 多工作表选择（Excel文件）
- 自定义数据范围选择
- 实时数据预览
- 智能表头识别

### 📈 图表类型
支持9种不同类型的交互式图表：
- **折线图** - 适合展示趋势变化
- **柱状图** - 适合比较数据大小
- **饼图** - 适合展示比例关系
- **散点图** - 适合展示相关性
- **雷达图** - 适合多维度数据对比
- **热力图** - 适合展示数据密度
- **树图** - 适合展示层级结构
- **漏斗图** - 适合展示流程转换
- **仪表盘** - 适合展示单一指标

### 🎨 界面特性
- 现代化响应式设计
- 三种布局模式：网格、垂直堆叠、水平排列
- 图表拖拽和缩放交互
- 图表下载功能
- 多图表同时显示
- 暗色主题支持

## 快速开始

### 1. 打开工具
直接在浏览器中打开 `index.html` 文件，无需安装任何软件或搭建服务器。

### 2. 上传数据
点击"选择文件"按钮，上传您的Excel或CSV文件。工具提供了示例文件 `sample_data.csv` 供您测试。

### 3. 配置数据
- 选择工作表（如果是Excel文件）
- 设置数据范围（开始行、结束行）
- 确认是否包含表头
- 点击"加载数据"

### 4. 生成图表
- 选择图表类型
- 选择X轴数据列
- 选择Y轴数据列（可多选）
- 输入图表标题
- 点击"生成图表"

### 5. 管理图表
- 使用布局管理调整图表排列
- 下载图表为PNG图片
- 删除不需要的图表

## 使用示例

### 示例数据说明
提供的 `sample_data.csv` 包含以下数据列：
- 月份：时间维度
- 销售额：主要业务指标
- 成本：支出指标  
- 利润：计算指标
- 客户数量：规模指标
- 满意度：质量指标

### 建议的图表组合
1. **趋势分析**：使用折线图展示销售额、成本、利润的月度变化
2. **对比分析**：使用柱状图比较各月份的关键指标
3. **比例分析**：使用饼图展示成本占销售额的比例
4. **相关性分析**：使用散点图分析客户数量与满意度的关系
5. **综合仪表盘**：使用仪表盘展示平均满意度

## 技术特点

### 纯前端实现
- 无需服务器，数据完全在本地处理
- 隐私安全，数据不会上传到任何服务器
- 跨平台支持，支持Windows、Mac、Linux

- **前端框架**：原生HTML5 + CSS3 + JavaScript ES6+

## 高级用法

### 多系列图表
选择多个Y轴数据列可以在同一图表中显示多个数据系列，适合进行对比分析。

### 自定义数据范围
通过设置开始行和结束行，可以只加载部分数据，适合处理大型数据集。

### 响应式布局
工具自动适配不同屏幕尺寸，在手机、平板、桌面设备上都有良好的显示效果。

### 图表导出
每个图表都支持导出为高清PNG图片，可用于报告制作和演示。

## 注意事项

1. **数据格式**：确保数据格式正确，数值列应包含纯数字
2. **文件大小**：建议单个文件大小不超过10MB
3. **浏览器内存**：生成大量图表时注意浏览器内存使用
4. **数据隐私**：所有数据处理都在本地进行，无任何网络传输

## 常见问题

**Q: 为什么我的Excel文件无法解析？**
A: 请确保使用标准的Excel格式(.xlsx或.xls)，避免使用过于复杂的格式和公式。

**Q: 图表显示不全或错位怎么办？**
A: 尝试调整浏览器窗口大小，或者切换不同的布局模式。

**Q: 可以处理多大的数据文件？**
A: 建议数据行数不超过10000行，以确保良好的性能表现。

**Q: 支持哪些数据类型？**
A: 支持数字、文本、日期等常见数据类型，图表主要基于数值数据生成。
